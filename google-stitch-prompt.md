# Google Stitch Prompt — aWARE by WARE AI — SaaS Landing Page

---

## Prompt:

Build a complete, polished single-page landing website for **aWARE** by **WARE AI** — a voice-activated AI desktop assistant SaaS product. The tagline is **"Effortless AI."** aWARE is an intuitive desktop voice assistant that listens, understands, and executes tasks for you — integrating with Google Workspace and Microsoft 365. It's sold as a B2B/B2B2C SaaS subscription targeting teams, operational leaders, project managers, sales teams, and fast-growing organizations.

---

### BRAND & COLOR PALETTE

- **Primary background:** Deep charcoal/near-black (#1A1A2E or #111111) — the page is predominantly dark mode.
- **Signature gradient:** Electric blue (#3B5BFF / #5B21B6 purple) → vivid green (#34D399), used on feature cards, CTA buttons, section accents. This blue-to-green gradient is the brand hallmark.
- **Accent text:** Bright blue (#2563EB) for highlighted keywords in headlines (e.g., "aWARE" in headings).
- **Voice command quotes:** Soft golden/yellow (#F6C547) for quoted user voice commands — these stand out as examples of what you can say.
- **Body text on dark:** Light gray (#9CA3AF) for readable paragraphs.
- **Card backgrounds:** Rounded-corner gradient cards (purple→blue→green) with 20–30px border-radius.
- **Typography:** Clean modern sans-serif (Inter or similar). Headlines are **very large and bold** (48–72px). Body text 16–18px.

### DESIGN EFFECTS & INTERACTIONS

1. **Large rounded gradient cards (20–30px border-radius)** — the signature visual element. Used for the product demo, productivity section, integrations, and contact form. The gradient always flows purple → blue → green.
2. **Dark mode throughout** — near-black background (#111 or #1A1A2E) is the default. There are NO white/light sections on this site.
3. **Smooth scroll-triggered fade-in animations** — elements fade up (translateY + opacity) as they enter the viewport.
4. **Circular photo frames** — workplace photos cropped into circles (for the boost productivity section) or large rounded rectangles.
5. **Golden/yellow quoted voice commands** — user voice examples are rendered in a warm golden-yellow italic style, visually distinct from body text.
6. **Gradient text on logo** — "WARE AI" uses CSS `background-clip: text` with the blue-to-green gradient.
7. **Purple pill CTA buttons** — solid purple (#5B21B6) with white text, arrow icon (→), hover: slight scale-up and glow.
8. **Ghost/outlined buttons** — white border, white text, transparent fill, used as secondary CTAs.
9. **App icon grid in dark circles** — integration logos sit in dark circular badges on gradient cards.
10. **Video play button overlay** — centered circle play icon on the demo screenshot.
11. **Hover effects** — buttons scale up slightly (1.03–1.05x) with a soft glow. Cards may have subtle hover lift.
12. **Responsive design** — single column on mobile, side-by-side on desktop.
13. **Large, confident typography** — headlines 48–72px, bold weight, creating a premium SaaS feel.
14. **No clutter** — generous whitespace (well, dark-space), breathing room between sections, minimal UI chrome.

---

### OVERALL TONE

Premium, dark, modern SaaS. Think **Linear, Vercel, or Raycast** landing page aesthetics — dark backgrounds, glowing gradients, large confident type, high-quality workplace photography. The page sells a productivity tool for professionals, so it should feel **smart, clean, and trustworthy** with social proof (HBR stat). Every section answers: "What does aWARE do for me?" through real voice command examples that show the product in action. No hardware, no devices — this is purely a desktop software assistant sold as a service.
